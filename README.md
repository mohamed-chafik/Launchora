# 🚀 Launchora - Rocket Launch Schedule

A modern, responsive web application that displays upcoming rocket launches using real-time data from The Space Devs API. Built with Next.js 15, TypeScript, and Tailwind CSS.

**[Launchora](https://launchora-rockets.vercel.app/)**

## ✨ Features

- **📅 Upcoming Launches** - Browse through scheduled rocket launches
- **🕐 Countdown Timers** - Real-time countdown to launch events
- **🖼️ High-Quality Media** - Stunning launch images and mission details
- **📱 Responsive Design** - Optimized for all devices
- **⚡ Blazing Fast** - Built with Next.js 15 for optimal performance
- **🎨 Modern UI** - Clean, space-themed interface with smooth animations
- **🔔 Real-time Updates** - Automatic data refresh and caching

## 🛠️ Tech Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **API**: The Space Devs API v2.3.0
- **Deployment**: Vercel
- **Fonts**: Geist Sans & Geist Mono
- **Icons**: Lucide React
- **UI Components**: Custom components with Tailwind
